Readme
-------

1. install-aks.sh  - Create new Azure AKS infrastructure
2. install-np.sh   - Create AKS node pools for your applications
3. pull-charts.sh  - Download helm charts to local folder (Optional). You may use these helm in install-base.sh 
4. install-base.sh - Install base software such as database, webservers, queing system.
5. install-app.sh  - Install your applications

6. overrides.env   - Environment customization or overrides
